### Install
```
I used yarn to install my packages and run scripts but npm will also work.

1 - $ git clone "https://github.com/BrahimS/weatherAPI.git"
2 - cd weatherAPI.
3 - $ yarn install or $ sudo yarn install
4 - $ yarn start

```
### Production

```
$ yarn build
